Capítulos del expediente

Cada cita activa abre un capítulo. Si la cita se cancela, el capítulo se elimina; si la sesión ocurre, puede quedar pendiente de documentar.

Capítulo clínico

Versión mínima indispensable: diagnóstico, autorización de seguro cuando aplique, nota clínica, observaciones y tareas. Esta es la versión WEB final estable.

Selecciona un capítulo para empezar.
Para pacientes de aseguradora, al documentar la sesión debes capturar el FOLIO DE AUTORIZACIÓN DE LA SESIÓN. En la primera sesión documentada también se exige diagnóstico si no existe uno previo.
Este folio cambia en cada sesión de aseguradora y es el dato que luego se usará para reportes.
